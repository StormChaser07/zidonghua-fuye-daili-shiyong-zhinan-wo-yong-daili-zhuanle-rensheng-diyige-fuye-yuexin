## ❓ 为什么自动化副业需要代理？

因为你不是“一个人在战斗”。一个脚本控制几十个账号、一台设备运行数十个任务，这些都会触发平台检测机制。

平台会判断：
- 你是不是同一IP登陆多个账号？
- 请求频率是否异常？
- 登录地是否频繁切换？

只要触发，就会封号、限制、打码验证。
